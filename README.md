# SplitDivision---master-backend

To run prettier:

```
npx prettier --write .
```

### Running the development server.

### To install

```bash
    npm install
```

### And run:

```bash
    npm run start
```

### To run project

#### Build docker image

```bash
    docker build -t master-backend .
```

#### Run a container on port 5000

```bash
    docker run -p 5000:5000 master-backend
```
